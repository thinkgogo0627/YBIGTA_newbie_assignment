여러분은 가상의 구독형 커머스 스타트업 **DailyBox**의 전략팀 분석가입니다. 신임 CMO(마케팅 총괄)가 다음 분기 마케팅 예산 배분을 여러분에게 맡겼습니다.

> "지난 분기 쿠폰 캠페인이 효과가 있었는지, 어떤 유입 채널이 진짜 돈 되는 고객을 데려오는지, 매출을 끌어올리는 진짜 레버가 뭔지 — 감(感) 말고 **데이터로 근거를 대주세요.** 마지막엔 한 장짜리 마케팅 전략 제안을 받고 싶습니다."

통계 검정 하나하나가 **"어떤 의사결정을 위한 근거인가"** 를 항상 생각하며 진행해주세요. 정답을 맞히는 것보다, 분석의 흐름을 경험하고 결과를 전략 언어로 해석해보는 것이 이번 과제의 핵심입니다.

## 명세

### 데이터

`marketing_customers.csv` — DailyBox 고객 6,000명의 한 달치 스냅샷입니다.

| 변수명 | 설명 | 유형 |
|---|---|---|
| customer_id | 고유 식별자 (분석 미사용) | - |
| age | 나이 | 수치형 |
| gender | 성별 (Female/Male) | 범주형 |
| signup_channel | 유입 채널 (Organic Search / Social Ads / Referral / Influencer) | 범주형 |
| membership_tier | 멤버십 등급 (Basic / Silver / Gold) | 범주형 |
| monthly_visits | 월 방문(접속) 횟수 | 수치형 |
| avg_discount_rate | 평균 적용 할인율 (0~0.5) | 수치형 |
| online_ad_cost | 고객 획득 온라인 광고비 | 수치형 |
| offline_ad_cost | 고객 획득 오프라인 광고비 | 수치형 |
| total_ad_cost | 총 광고비 (= online + offline) | 수치형 |
| campaign_group | 지난 분기 쿠폰 캠페인 배정 (control=미발송 / treatment=발송) | 범주형(2집단) |
| responded | 캠페인 반응(전환) 여부 (0/1) | 범주형(이진) |
| monthly_spend | **월 구매액(매출) — 핵심 분석 대상** | 수치형 |

- 결측치는 없습니다. `customer_id`는 분석에 사용하지 않습니다.
- `total_ad_cost = online_ad_cost + offline_ad_cost` 로 **정의상** 계산된 값입니다 (10번에서 직접 다룹니다).
- `campaign_group`은 고객을 무작위로 절반씩 나눠 한쪽(treatment)에만 쿠폰을 보낸 결과입니다. 두 그룹이 무작위로 나뉘었으므로, **쿠폰 발송군 vs 미발송군**의 매출 차이를 캠페인 효과로 해석할 수 있습니다.

### Part 1. 고객 이해 

**1. 데이터 로드 및 구조 확인**
아래 코드로 데이터를 불러오고 구조를 확인하시오.
```python
import pandas as pd
df = pd.read_csv('marketing_customers.csv')
```
`shape`, `dtypes`, `describe()`, 변수별 결측치 개수를 출력하고, 고객 규모와 주요 수치 변수의 범위를 한두 줄로 요약하시오.

**2. 매출 분포와 로그변환**
`age`, `monthly_visits`, `monthly_spend`의 분포를 히스토그램으로 그리시오. 특히 `monthly_spend`의 **왜도(skewness)** 를 계산하고, `log1p(monthly_spend)` 변환 전/후 분포를 비교 시각화하시오.
- 매출이 왜 이렇게 치우쳐 있는지("소수 고가치 고객에 매출이 쏠려 있는가?"), 로그변환이 왜 도움이 되는지 간단히 서술하시오.

**3. 채널·등급별 매출 시각화**
`signup_channel`별, `membership_tier`별 `monthly_spend`를 각각 Boxplot으로 비교하고, 어떤 채널/등급이 고가치 고객으로 보이는지 간단히 해석하시오.

### Part 2. 가정 확인

**4. 1종/2종 오류 (개념)**
"쿠폰 캠페인은 매출에 효과가 없다"는 귀무가설을 예로 들어, **1종 오류**와 **2종 오류**가 이 비즈니스 상황에서 각각 무엇을 의미하는지, 각각 회사에 어떤 손실(예산 낭비 vs 기회 손실)을 끼치는지 서술하시오.

**5. 정규성 검정 (Shapiro-Wilk)**
한 유입 채널(예: `Organic Search`)의 `monthly_spend`가 정규분포를 따르는지 Q-Q plot과 Shapiro-Wilk 검정으로 확인하시오. `log1p` 변환 후 같은 검정을 반복해 결과를 비교하시오. (가설 수립 포함, 유의수준 0.05 기준 해석)
> (단, 이후 분석은 정규성을 만족한다고 가정한다.)

**6. 등분산성 검정 (Levene)**
4개 유입 채널 간 `monthly_spend`의 분산이 동일한지 Levene 검정으로 확인하고, p-value를 0.05 기준으로 해석하시오. 이 결과가 이후 분산분석에서 어떤 의미를 갖는지 서술하시오.

### Part 3. 집단 비교 검정

**7. 쿠폰 캠페인, 효과 있었나? — 두 집단 평균 비교**
`campaign_group`이 `treatment`인 고객과 `control`인 고객 간 `monthly_spend` 평균에 유의한 차이가 있는지 검정하시오.
- 먼저 Levene 검정으로 등분산 가정을 확인하고, 만족되지 않으면 **Welch's t-test**를 사용하시오.
- H0/H1을 명시하고 결론을 서술하시오. **다음 분기에도 캠페인을 계속할 것인가?**

**8. 채널 비교 — One-way ANOVA + Tukey HSD**
`signup_channel`(4그룹)에 따라 `monthly_spend` 평균에 차이가 있는지 One-way ANOVA로 검정하시오. (가설 수립 포함)
- 유의하면 **Tukey HSD** 사후검정으로 어떤 채널 쌍에 차이가 있는지 정리하시오.
- Boxplot과 종합하여 **어느 채널이 통계적으로 유의하게 고가치/저가치 고객을 데려오는지** 결론지으시오.

**9. 교호작용 — Two-way ANOVA (선택 심화)**
`signup_channel`과 `campaign_group`이 `monthly_spend`에 미치는 영향을 Two-way ANOVA로 검정하시오. 각 주효과와 **채널 × 캠페인 교호작용**이 유의한지 확인하고, "캠페인 효과가 채널마다 다른가?"를 해석하시오.

### Part 4. 회귀분석 — 매출을 끌어올리는 진짜 레버는?

**10. 단순선형회귀 & 다중회귀**
- (단순) `monthly_visits` → `monthly_spend` 단순선형회귀를 적합하고 회귀계수·R²을 해석하시오.
- (다중) 종속변수 `monthly_spend`, 독립변수 `monthly_visits` + `avg_discount_rate` + `online_ad_cost` + `signup_channel`(더미) + `gender`(더미)로 다중회귀를 적합하시오. statsmodels `ols()`와 `C(...)` 더미코딩을 사용하고, 각 계수를 **전략 관점**에서 해석하시오(방문·할인·광고비 중 매출에 실제로 기여하는 레버는? 할인 확대는 좋은 전략인가?).
- R²·Adjusted R²을 단순회귀와 비교하시오.

**11. 다중공선성 / VIF**
`online_ad_cost`, `offline_ad_cost`, `total_ad_cost` **세 변수를 모두** 넣어 VIF를 계산하시오. VIF에 어떤 문제가 나타나는가? 왜 그런가(`total = online + offline`)? `total_ad_cost`를 뺐을 때 VIF가 어떻게 달라지는지 비교하고 해결책을 서술하시오.
> (Hint: `sm.add_constant(X)` → 컬럼별 `variance_inflation_factor(X.values, i)`. 보통 VIF>10이면 심각.)

**12. 잔차 진단**
다중회귀 모형의 잔차에 대해 정규성(Q-Q plot), 등분산성(잔차 vs 예측값 산점도)을 시각화해 회귀 가정을 점검하시오. 가정이 깨져 보인다면 종속변수를 `log1p(monthly_spend)`로 바꾼 모형을 추가로 적합해 잔차가 개선되는지 비교하시오.

### Part 5. 타겟팅 (로지스틱 회귀)

**13. 로지스틱 회귀**
`responded`를 종속변수로, `age` + `monthly_visits` + `campaign_group`(더미) + `membership_tier`(더미) + `signup_channel`(더미)을 독립변수로 하는 로지스틱 회귀를 적합하시오.

**14. 오즈비 해석 & 타겟팅**
유의한 변수를 확인하고 계수를 **오즈비(`exp(coef)`)** 로 변환해 해석하시오. 특히 **쿠폰 발송 여부와 멤버십 등급**이 반응 확률에 미치는 영향을 서술하고, "다음 캠페인은 어떤 고객군에 집중해야 반응률이 높을지" 결론지으시오.
> (Hint: 오즈비>1이면 반응 오즈 증가, <1이면 감소. 예: 2.4면 반응 오즈가 2.4배.)
